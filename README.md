## 📊 Project Overview
An interactive Excel dashboard analyzing employee attrition patterns using the IBM HR Analytics dataset. This project transforms raw HR data into actionable insights through dynamic visualizations and interactive filters.

## 🎯 Key Features
- **Interactive Dashboard** with connected slicers for real-time filtering
- **Key Metrics**: Total employees (1,470), Attrition rate (16%), Avg age (37), Avg income ($6,503)
- **Multi-dimensional Analysis**:
  - Attrition by Department
  - Overtime Impact Analysis
  - Job Satisfaction of Leavers by Role
  - Income Comparison (Stayers vs Leavers)
  - Age & Gender Attrition Patterns

## 🛠️ Tools Used
- **Microsoft Excel** (PivotTables, PivotCharts, Slicers, Formulas)
- Data Cleaning & Preparation
- Interactive Dashboard Design

## 🎨 Interactive Filters
Dynamic slicers allow filtering by:
- Department (HR, R&D, Sales)
- Gender (Male, Female)
- OverTime (Yes, No)
- JobRole (9 roles including Manager, Scientist, Executive)

## 🔍 Key Insights Uncovered
1. **Sales department** has the highest attrition rate
2. Employees working **overtime** are significantly more likely to leave
3. **Job satisfaction** strongly correlates with retention
4. **Income gap** exists between those who leave and those who stay
5. **26-35 age group** shows highest attrition, particularly males
